# Browsers on Windows 7

Several modern-ish browsers still run on Windows 7, either natively or through VxKex.

## Opera 95

**Setup:**
1. Download the standard Opera 95 installer from Opera's official site or a trusted archive if the current site only serves a newer version.
2. Run the installer directly — no VxKex required.
3. Complete setup normally; sign into your Opera account if desired (optional).

## Vivaldi 3.7

**Setup:**
1. Search "Vivaldi 3.7 installer" — the official Vivaldi site typically only offers the latest version, so use their older-releases archive page if available, or a trusted mirror.
2. Confirm the version is exactly 3.7 before installing — later versions drop Win7 support.
3. Run the installer directly — no VxKex required.
4. After install, check Settings → About to confirm it didn't auto-update to a newer, incompatible version; disable auto-update in Settings if the option exists.

## Pale Moon

**Setup:**
1. Download Pale Moon from the official palemoon.org site — it actively supports older Windows versions, so no special version-hunting needed.
2. Run the installer directly — no VxKex required.
3. Works out of the box; no extra compatibility steps needed.

## Midori (Gecko-based build)

**Setup:**
1. Search specifically for "Midori 11.0.0 Gecko" — there are two different Midori codebases (an older WebKit-based one and this newer Gecko-based one), so confirm you're getting the Gecko build.
2. Run the installer directly — no VxKex required.
3. No further compatibility steps needed after install.

## Recommendation

For a low-RAM Win7 machine: Pale Moon or Midori 11.0.0 for lightest resource usage; Opera 95 or Vivaldi 3.7 if you need closer-to-modern Chromium compatibility.
